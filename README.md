# wow-addon

Collection d'addons WoW Midnight (12.0).

## Addons

| Addon | Description |
|-------|-------------|
| [WannaQueue/](WannaQueue/) | Auto-accept les role checks LFG |
| [PIGlow/](PIGlow/) | Glow raid frames quand quelqu'un MP pour Power Infusion (universel) |
| [PIGlowDander/](PIGlowDander/) | Version legere de PIGlow, exclusivement pour Dander Frames |

## Installation

Chaque dossier est un addon independant. Pour installer un addon :

1. Copier le dossier de l'addon dans `World of Warcraft/_retail_/Interface/AddOns/`
2. Relancer WoW ou `/reload`
