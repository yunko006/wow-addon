# WannaQueue

Auto-accept les role checks LFG sur WoW Midnight (12.0). Quand le chef de groupe lance une queue, l'addon accepte automatiquement le popup de confirmation de role.

Equivalent du WeakAura : https://wago.io/HyHjIHeKm

## Installation

1. Copier le dossier `wannaqueue/` dans `World of Warcraft/_retail_/Interface/AddOns/`
2. Relancer WoW ou `/reload`

## Utilisation

| Commande | Action |
|----------|--------|
| `/wq` | Toggle l'auto-accept on/off |
| `/wannaqueue` | Alias de `/wq` |

L'addon est actif par defaut au premier chargement.

## Compatibilite

- WoW Midnight 12.0 (Interface 120001)
